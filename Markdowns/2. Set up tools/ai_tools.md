## When AI Is Most Useful for Coding
I think AI is most useful when:
- Writing repetitive or boilerplate code
- Getting unstuck on small bugs or errors
- Learning new technologies or concepts
- Reviewing or improving existing code

AI is less useful when making architectural decisions, working with sensitive data, or writing critical logic without human review.
