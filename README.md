# React + Tailwind CSS Setup

## Overview
This project was set up using React and Tailwind CSS to create a modern, utility-first styling workflow.

## Setup Steps
1. Created a React project using Vite.
2. Installed Tailwind CSS, PostCSS, and Autoprefixer.
3. Configured Tailwind to scan React files.
4. Added Tailwind directives to the global CSS file.
5. Verified Tailwind by applying utility classes in a React component.

## Running the Project
```bash
npm install
npm run dev
